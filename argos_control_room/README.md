# Argos Control Room (Electron + React + TypeScript + Tailwind)

Desktop control-room interface for the Argos system.

## Stack

- Electron main/preload process
- React renderer with TypeScript
- Tailwind CSS for styling
- Vite for renderer build/dev

## Run

1. Install dependencies:
   npm install
2. Development mode (Vite + Electron):
   npm run dev
3. Production-style local run (build then Electron):
   npm run start

## Checks

- Type and process checks:
  npm run check
- Build renderer bundle:
  npm run build
